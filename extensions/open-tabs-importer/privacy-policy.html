Privacy Policy

SmartTools Open Tabs Importer

This Chrome/Edge extension is designed to import your currently open browser tabs into your personal SmartTools bookmark manager.

Data Collection

The extension collects the title and URL of your currently open browser tabs only when you actively click the import button. No data is collected, transmitted, or stored remotely by the extension itself.

All imported data is sent directly to your own SmartTools backend instance for storage — it does not go through any third-party server controlled by this extension.

Data Usage

  • Read open tabs (titles and URLs) only at the user's explicit request
  • Send tab data to your own SmartTools backend via window.postMessage
  • Store the SmartTools backend URL locally in browser chrome.storage (local to your browser only)

Data Storage

The extension stores the following locally in your browser — no remote server is involved:

  • The configured SmartTools backend URL (chrome.storage.local)
  • Pending import data during the import handshake (chrome.storage.local, cleared after delivery)
  • Category preference for import destination (cookie, scoped to your SmartTools domain)

Permissions Used

  • tabs — read tab titles and URLs for the import list
  • scripting — inject the import handshake script into the SmartTools config page
  • storage — save the backend URL and pending import state locally
  • — required to deliver import data to your SmartTools backend (which may be hosted at any domain you configure). The content script only performs work on your own SmartTools backend page.

Data Sharing

The extension does not share any user data with third parties, analytics services, or ad networks.

Contact

If you have questions about this privacy policy, please open an issue on the project GitHub repository.
