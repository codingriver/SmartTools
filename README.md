# SmartTools - 自用在线小工具集

一些自用的网址链接收藏，以及随手写给自家小朋友使用的在线小工具，已有的工具如下：

## 估算计算器

一款专门用于训练数学估算能力的智能计算器，帮助孩子系统掌握数学估算技巧。

**主要功能：**
- 支持加减乘除四则运算的估算
- 提供多种估算方法（首位估算法、整十数法、次位取整法等）
- 显示详细的估算过程和步骤说明
- 同时显示精确结果和估算结果供对比
- 支持键盘输入，操作简便

**特色：**
- 智能选择最优估算结果，提供多个候选估算值
- 详细的估算步骤说明，便于理解和学习

## 诗词九宫格

通过九宫格游戏的方式帮助孩子记忆和学习古诗词，寓教于乐。

**主要功能：**
- 内置1-6年级上下册诗词库（按每周一首编排）
- 随机诗词功能（调用今日诗词API）
- 自动生成九宫格汉字游戏，点击汉字可随机替换
- 提供完整诗词答案查看，支持百度搜索诗词详细信息
- 支持自定义输入诗词内容

**特色：**
- 诗词库按年级和学期系统整理
- 九宫格汉字随机打乱，增加趣味性
- 游戏化学习方式，提高记忆效果
- 界面简洁美观，适合小朋友使用

## 生理学知识点大纲

系统整理人体生理学各章节知识要点，结合临床实际应用的专业学习工具。

**主要功能：**
- 涵盖细胞生理、循环系统、呼吸系统等12个核心章节
- 四级知识点结构，从基础概念到具体机制层层深入
- 每个知识点配备"主要关联知识点"模块，建立系统性联系
- 提供详细的临床应用实例和疾病对照说明
- 支持章节展开/收起功能，便于针对性学习

**特色：**
- 知识点按教学逻辑系统编排，符合生理学学习规律
- 临床关联丰富，每个生理机制都对应实际疾病案例
- 突出病理生理联系，帮助理解正常功能与疾病的关系
- 专业术语准确，适合医学生和医护人员参考学习

## AI 错题本

基于 AI 的智能错题管理工具，拍照识别、自动解析、举一反三，让错题真正变成提分利器。

**主要功能：**
- 支持拍照/截图自动识别题目，AI 生成详细解析和同类练习题
- 多科目错题本管理，智能提取知识点标签
- 多维度筛选查看，可视化学习进度统计
- 兼容 Gemini 和 OpenAI 接口
- 支持一键导出错题打印，方便线下复习

**特色：**
- AI 驱动的举一反三，自动生成同类变式题巩固薄弱点
- 支持多用户独立使用，具备 PWA 离线访问能力
- 知识点标签自动归类，错题规律一目了然

## 密码管理器

安全的在线密码生成与管理工具，所有操作均在浏览器本地完成，不上传任何数据。

**主要功能：**
- 支持自定义密码长度和字符类型组合（大小写字母、数字、特殊符号）
- 一键生成高强度随机密码
- 实时密码强度评估与反馈

**特色：**
- 纯本地运算，隐私安全有保障
- 界面简洁直观，即开即用

## 影视聚合

主要基于 [MoonTV](https://github.com/gagabba/MoonTV-gas) 的影视聚合播放器，无需独立服务器，免费轻量运行。

**主要功能：**
- 内置多个免费资源站点，一次搜索返回全网影视结果
- 支持在线流畅播放，自动去除广告
- 收藏与播放记录云端同步
- 响应式布局，适配手机与桌面端

**特色：**
- 多源聚合搜索，告别逐站找片

## 每日数学练习

自动生成小学生数学每日练习题目，包含题目和标准答案，支持直接打印。

**主要功能：**
- 自动生成口算练习题
- 竖式计算题、脱式计算题及应用题
- 同时生成对应的完整答案
- 打印友好的页面设计

**特色：**
- 题目难度符合小学教学要求
- 题型丰富，涵盖基础计算到复杂运算
- 答案包含完整的竖式计算过程和解题步骤
- 支持直接打印，方便日常练习

## 在线收藏夹

自用网站链接收藏。

---

这些工具都采用响应式设计，支持手机和电脑使用，无需安装，打开即用。
